KenWeTech's Paperback Themes

Themes are organized by color or theme name. Each theme folder contains available variants.

  • Standard version: [Theme-Name]-Paperback.pbcolors
  • BDM: [Theme-Name]-BDM-Paperback.pbcolors
    Black Dark Mode version with a pitch-black dark mode background, optimized for OLED displays.
  • BAM: [Theme-Name]-BAM-Paperback.pbcolors
    Black All Modes version with a pitch-black background across both light and dark modes, optimized for OLED displays.
